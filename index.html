关于健身

发表于 2018-03-19 | 分类于 健身 |

开始

我几乎每天都会去刷一下朋友圈，看看自己的朋友都晒啥东西，秀恩爱的、晒孩儿的、代购、好吃的，干啥的都有，于是感叹大家的生活真是丰富啊。突然看到朋友圈有晒肌肉的，看到别提有多眼馋了！

有一天我跟章鱼星老哥聊天，那个时候他就是一个坚持健身5年多的老司机了。跟他聊天也让我对健身充满了憧憬，也暗暗下定决心开始健身，虽然不至于变成一个肌肉侠，但是身材单薄，没有看点的我，还是有那么一点点的自卑。我也是突然地、强烈地想要肌肉，我要变强壮，我要肌肉！！

办健身卡

那个时候我在裕惠大厦上班，而恰好在我公司的楼下就有个健身房，这可能也是促使我健身的一个条件，有这么便利的条件还不好好利用对得起公司这么好的选址吗！

于是第二天到公司楼下B1层健身房用¥1710办了一年的健身卡。就这么的，开始了我的健身事业，健身房不大，有8个跑步机，6个椭圆机，健身自行车，一个大哑铃架，两个杠铃架，还有一些其他的器材，反正是足够我这个菜鸟用了。

而雷哥最近也有办健身卡的打算，于是我们两个就一起开始了为期一年的搭伴吃喝虐肌肉。

坚持就是胜利

从办健身卡的那天开始，基本保持每周一三五去健身，周日选择性的去练。期间玩跑步机的时候崴过脚，休养了一段时间。

健身一个月之后，慢慢的有了效果，看着自己慢慢变紧实的肌肉，慢慢变大的胸肌，心里真是特别的高兴，每次健身完都会看着镜子里自己肌肉的线条自恋一会儿。每次都有充实的满足感。

而在举铁的时候有时会产生懈怠的想法，想着快做完去休息一下，或者本来应该做12个，做了10个就停了，这个时候偷懒就是自己骗自己，貌似骗自己真不会害羞啊，不过在意识到自己的不妥之后就改掉了，毕竟做了决定就要对得起自己。认真的完成每一个动作，认真的感受每一个动作对肌肉的撕扯才能起到最好的效果，否则懈怠的情绪很容易让动作变形，别的肌肉会参与到这个动作的完成，这样本应该受到锻炼的肌肉就得到了放松的机会，达不到预期的效果了。

印象中在17年2月14情人节的晚上，公司组织去电影院看《极限特工》。刚开场在我还没酝酿好情绪的时候，范·迪塞尔踩着滑板从山顶滑下来，紧张刺激的场景瞬间就抓住了我的注意力，让我瞬间就融入到场景中，心跳不自觉的在加快，可能是在肾上腺素的影响下，他那强壮的肌肉瞬间就征服了我。偶像，必须偶像，简直太帅了。我一定一定要坚持下去，我要成为mini范·迪塞尔！！

整整一年的变化

健身一整年，也就是从16年4月——17年4月，在健身卡到期的这天，健身完站在镜子前跟一年前的自己对比，发现自己变化了很多，脸部貌似也有了棱角，肌肉的线条也更加明显了，也越来越自信了，健身对自己的心理影响也是潜移默化的。

而这次健身也让我酷爱肌肉，虽然健身卡到期了，但是家里也有哑铃，自己在家也各种动作的练习。

为了锻炼腹肌，买了个瑜伽垫，尝试着跟着视频学习做肌肉撕裂者，每天都感觉自己的腹肌在抽筋，难受自然是很难受，但是成果还是很明显的，腹部的脂肪也是减少了很多，肌肉块也慢慢的展现出来了。

在健身的过程中我发现自己超级喜欢肌肉酸疼的感觉！健身过后肌肉酸疼的感觉能让我明显的感受到肌肉的存在，而我把这个当成是劳累半天的收获，或者说是回馈。

之所以停下来

17年5月份发生了一些事，对我当时的心境影响很大，一方面读《人类简史》，发现自己对于人类的起源真是毫无所知，进而也产生了一些迷茫的情绪，那就是人类要到哪去，我要到哪去，我生活的意义是什么，我为什么活着呢？迷茫恐惧无助的情绪，在那短时间彻底吞噬了我，再加上工作上遇到了一些挫折，让我每天都处于一种很煎熬的状态。于是我想是不是应该放松一下自己，不要让自己处于一种很紧张，心里很疲惫的状态。

正好自己也一直计划着一场骑行，路线是从北京到老家齐齐哈尔，全程1500公里左右。于是决定趁着这个机会，让自己的心去透透气，否则在这么下去我非得抑郁不可。于是我决定辞职，然后去完成自己的这个计划。6月中旬从公司离职，准备了一周就开始了一场说走就走的骑行，当然这一路很艰辛，不过这就是另一个主题了。所以健身就耽搁下来了，而这一停就是半年多。

健身是一辈子的事业

17年10月中旬回到了北京，4个多月没上班了，于是开始学习准备接下来的面试，又看了一遍《图解TCP/IP》，虽然面试的时候几乎没有问这个的，我这次看它的目的就是给自己增加信心。然后看了一些算法题。11月中旬开始面试，12月初入职。虽然没面试几家，但是还是得到了自己梦想中公司的offer，还是很开心的，虽然工资低了一些，但为了梦想嘛，值了！（虽然进去之后发现与预期天差地别）。

回北京的时候，自己还是处于一种异常的状态，感冒好长时间才好，总是头疼，也不知道是不是焦虑太大导致的，这种情况现在也偶尔会出现，怕不是烧坏了脑子了吧。过度的焦虑让我很煎熬，身体机能也跟着急速的下降。虽然说后来让自己走出了焦虑的阴霾，但是这期间的滋味真是不好受，身体也受到了一些损伤。

于是规律的健身又重新提上了日程，先从每次200个俯卧撑开始。而这次坚持多久我也不知道，我希望就这么一直下去吧，不存在坚持，把健身当成每天的必修课，把它当成吃饭喝水就好了。

加油吧！少年。年轻的你怎么知道未来在何方，焦虑又能解决什么问题，脚踏实地一步一个脚印，哪怕明天就不做程序员了，今天也不能放下这本Code秘籍。

技术选择

Github
Git
Nodejs
Hexo

都是封装好的

在开始各种安装之前，最好还是简单的理解一下各个部分是干什么用的，要不一会安装这个一会安装那个，这种迷迷糊糊不受控制的感觉可能并不是太好，只有先从整体上了解它，才能对各个部分有一个把控，这也是我一直践行的一个方法论，先整体后局部。

  • Github :代码托管的地方，以这个博客工程为例，我们写的博客最后生成的是静态页面html文件，这些静态页面html文件就放在这个地方。

  • Git : 分布式版本控制技术，多人协作的时候就需要用到版本控制了，因为只有这样才能让多人同时操作一个文件，github就是基于git建立的一个代码托管平台，使用各种git命令就能与github上的代码库进行通信。

  • Nodejs : 扮演的是一个服务器的角色，只不过这个服务器并不是要与客户端通信的那种服务器。hexo就运行在其上。

  • Hexo : 它是基于nodejs的静态网页框架，同时它封装了对git的操作，也就是说Git和Nodejs其实是它的底层服务。使用hexo的几个命令就能完成文章在github的部署。正是得益于它对git的封装。

//自己的思路，正在考证

可能会有疑问，为什么配这个东西，像代码这种可能会涉及机密的数据，在传输的时候一定要使用一定的加密手段，如果直接使用明文传输很有可能数据传输在传输的过程中被非法获取。在众多加密手段中有一种是非对称加密，在数据传输的过程中需要用到两个密钥（公钥和私钥）它们是成对出现的，公钥是用来加密的，而私钥是用来解密的，用这个公钥进行加密必须使用与之配对的私钥进行解密。公钥发布出来，而私钥自己保留，我们在与其他人传输数据的时候使用对方给我们的公钥进行加密，数据到达对方时，它会使用自己的私钥进行解密，即便第三方获取了数据，由于他们没有与加密时使用的公钥配对的私钥，他们也不可能获取到数据。以此来保证数据传输的安全性。同样对方给我们传输数据的时候使用我们的公钥进行加密，我们使用自己的私钥进行解密。这个私钥是相当私密的，是只属于自己的，绝对不能让任何人知道。而我们给github配置的这个就是我们的公钥的，github会使用它来加密传递给我们的数据。

关于博客

发表于 2018-03-18 | 分类于 搭建博客 |

开始

作为博客的第一篇，就不写心情不写技术啦，聊聊关于博客的这些事儿吧。其实很久之前就一直想弄一个自己的博客站，也用Django自己搭建过一个博客，只不过比较粗糙。前端要学习的东西还是很多的，再加上需要很强的设计功底，设计样式、设计配色、设计动画。所以只是弄了个大概，再加上从北京离开半年，博客工程也就断了，实在是拿不出手。

索性就用人家的框架搭了一个，后续会把简易的搭建过程及后续的博客进阶方法总结出来，也可以说是随着自己的深入学习，把学习心得总结出来。

我一直深刻的记得高中班主任说过的话，知其然知其所以然。我也时刻用这句话警醒自己，在我学习技术的时候，它也对我产生了非常积极的影响。在学习一些框架的时候，也总是想着去掌握它的原理，它背后隐藏的思考，这才是学习一个东西更应该掌握的，而不只是简单的使用其api。

老师想表达的是，我们不仅要学习这个知识点，还有学习这个知识点背后的思考。

思考 很重要，读书有益，但是学而不思则罔，只是一本本的去读书，但并不对它进行思考，读完放在一边，那么读书只是枉然，只是满足了自己的虚荣心——我又看完了一本书，而书中的精华思想，是否真的了解了呢，或者说作者想要表达给我们的东西是什么呢？

意义

  • 好记性不如烂笔头

哪怕写下笔记之后不会再复习，印象也比只看一遍深得多。而后来回头看笔记的话收获绝对比第一次学习要收获更多。而如果是精心整理的笔记那不论是写笔记的当时，或者复习的时候，收获都是比只看一遍要强的太多太多。

过去的一段时间我有些浮躁，书看的是不少，但大都是比较快速的阅读，这得到的可能只是皮毛，更精华的部分被无意间忽略了。

  • 总结形成知识体系

上学的时候，记笔记好像是很自然的事，源头可能是上初中的时候写英语笔记。而大部分是将老师说的东西当成真理写下来记下来。高中的时候写笔记真的是经过自己思考写下来的，我的生物化学成绩一直很好，应该得益于我将那几本书从头到尾的看了一遍，仔仔细细的记了笔记，形成了知识体系。而知识体系对于举一反三，对于接受更深层次的知识有着至关重要的作用。

而很多学习过的知识，过后用到了也是模模糊糊，还得再去查一遍，从众多的文章中找到自己想要的那个，非常的影响效率。在学习的时候本来可以写下来的，然后随着学习的深入，随着自己对于相关问题的理解能力的提升，慢慢的丰富这个知识点，从而达到对一个知识点、一个知识体系的完全掌握。

  • 精心设计反复琢磨

一篇好的博客并不是一蹴而就的，而是先有一个主题，然后围绕着这个主题定一个骨架，而后整理各种材料，取其精华，将各路大神的想法思路，在自己的大脑中整理清楚，然后写出来的。

而往往写完之后还会校验几遍，是否有错别字，语义表达是否含混不清。自己的表达是否够清晰简洁。所以说一篇好的博客必定是消耗作者很多精力写成的。

在写博客或者说写笔记的过程中，反复思考应该是这个过程中对我们最有益了。用进废退，大脑只有多思考才能更聪明清楚有条理。

  • 宁心静神与己沟通

写博客是一个与自己沟通的过程，而写的越多思路就会越清晰，与别人沟通也会能更容易的表达自己的想法，能更快的让别人找到自己说话的重点在哪。如果只是自己能懂那是最下乘的文章，如果同行能懂算是中等的文章，外行人能看懂那就是一篇上乘的文章，而如果所有人都能很愉快的读完并从中有所收获，那就是上上乘的文章。我平时也遇到过一些上上乘的文章，这里任何一篇都是精品，让我有很轻松的阅读体验。

坚持

以前一直没有开始成规模的写博客，这可能是一个很枯燥的过程。胆子小又是处女座的我一直不敢轻易的做一些事，但是既然做了就要好好的做。而我的耐心也是要好好的磨练一下了。也不知道从什么时候开始，浮躁的不行。静不下心怎么能行。

印象笔记倒是很雄厚了，各种技术博客的收集，日记、随笔，倒是也写了不少，但是一方面没公开的东西自己写的也并不是全力以赴（一遍成，并没有精心整理），本可以写的更好的，所以开博客督促自己精心的去整理笔记。

我相信熟能生巧，量变会引发质变。大量的重复学习才能彻底掌握一个东西，我之前学过一段时间的python，但是快一年不接触了，很多东西都忘了。要是当时写下笔记，现在复习一下，或许能更快的捡起来了吧。马后炮啊~~
