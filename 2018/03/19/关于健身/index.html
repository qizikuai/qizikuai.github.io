关于健身

发表于 2018-03-19 | 分类于 健身 | 阅读次数:

开始

我几乎每天都会去刷一下朋友圈，看看自己的朋友都晒啥东西，秀恩爱的、晒孩儿的、代购、好吃的，干啥的都有，于是感叹大家的生活真是丰富啊。突然看到朋友圈有晒肌肉的，看到别提有多眼馋了！

有一天我跟章鱼星老哥聊天，那个时候他就是一个坚持健身5年多的老司机了。跟他聊天也让我对健身充满了憧憬，也暗暗下定决心开始健身，虽然不至于变成一个肌肉侠，但是身材单薄，没有看点的我，还是有那么一点点的自卑。我也是突然地、强烈地想要肌肉，我要变强壮，我要肌肉！！

办健身卡

那个时候我在裕惠大厦上班，而恰好在我公司的楼下就有个健身房，这可能也是促使我健身的一个条件，有这么便利的条件还不好好利用对得起公司这么好的选址吗！

于是第二天到公司楼下B1层健身房用¥1710办了一年的健身卡。就这么的，开始了我的健身事业，健身房不大，有8个跑步机，6个椭圆机，健身自行车，一个大哑铃架，两个杠铃架，还有一些其他的器材，反正是足够我这个菜鸟用了。

而雷哥最近也有办健身卡的打算，于是我们两个就一起开始了为期一年的搭伴吃喝虐肌肉。

坚持就是胜利

从办健身卡的那天开始，基本保持每周一三五去健身，周日选择性的去练。期间玩跑步机的时候崴过脚，休养了一段时间。

健身一个月之后，慢慢的有了效果，看着自己慢慢变紧实的肌肉，慢慢变大的胸肌，心里真是特别的高兴，每次健身完都会看着镜子里自己肌肉的线条自恋一会儿。每次都有充实的满足感。

而在举铁的时候有时会产生懈怠的想法，想着快做完去休息一下，或者本来应该做12个，做了10个就停了，这个时候偷懒就是自己骗自己，貌似骗自己真不会害羞啊，不过在意识到自己的不妥之后就改掉了，毕竟做了决定就要对得起自己。认真的完成每一个动作，认真的感受每一个动作对肌肉的撕扯才能起到最好的效果，否则懈怠的情绪很容易让动作变形，别的肌肉会参与到这个动作的完成，这样本应该受到锻炼的肌肉就得到了放松的机会，达不到预期的效果了。

印象中在17年2月14情人节的晚上，公司组织去电影院看《极限特工》。刚开场在我还没酝酿好情绪的时候，范·迪塞尔踩着滑板从山顶滑下来，紧张刺激的场景瞬间就抓住了我的注意力，让我瞬间就融入到场景中，心跳不自觉的在加快，可能是在肾上腺素的影响下，他那强壮的肌肉瞬间就征服了我。偶像，必须偶像，简直太帅了。我一定一定要坚持下去，我要成为mini范·迪塞尔！！

整整一年的变化

健身一整年，也就是从16年4月——17年4月，在健身卡到期的这天，健身完站在镜子前跟一年前的自己对比，发现自己变化了很多，脸部貌似也有了棱角，肌肉的线条也更加明显了，也越来越自信了，健身对自己的心理影响也是潜移默化的。

而这次健身也让我酷爱肌肉，虽然健身卡到期了，但是家里也有哑铃，自己在家也各种动作的练习。

为了锻炼腹肌，买了个瑜伽垫，尝试着跟着视频学习做肌肉撕裂者，每天都感觉自己的腹肌在抽筋，难受自然是很难受，但是成果还是很明显的，腹部的脂肪也是减少了很多，肌肉块也慢慢的展现出来了。

在健身的过程中我发现自己超级喜欢肌肉酸疼的感觉！健身过后肌肉酸疼的感觉能让我明显的感受到肌肉的存在，而我把这个当成是劳累半天的收获，或者说是回馈。

之所以停下来

17年5月份发生了一些事，对我当时的心境影响很大，一方面读《人类简史》，发现自己对于人类的起源真是毫无所知，进而也产生了一些迷茫的情绪，那就是人类要到哪去，我要到哪去，我生活的意义是什么，我为什么活着呢？迷茫恐惧无助的情绪，在那短时间彻底吞噬了我，再加上工作上遇到了一些挫折，让我每天都处于一种很煎熬的状态。于是我想是不是应该放松一下自己，不要让自己处于一种很紧张，心里很疲惫的状态。

正好自己也一直计划着一场骑行，路线是从北京到老家齐齐哈尔，全程1500公里左右。于是决定趁着这个机会，让自己的心去透透气，否则在这么下去我非得抑郁不可。于是我决定辞职，然后去完成自己的这个计划。6月中旬从公司离职，准备了一周就开始了一场说走就走的骑行，当然这一路很艰辛，不过这就是另一个主题了。所以健身就耽搁下来了，而这一停就是半年多。

健身是一辈子的事业

17年10月中旬回到了北京，4个多月没上班了，于是开始学习准备接下来的面试，又看了一遍《图解TCP/IP》，虽然面试的时候几乎没有问这个的，我这次看它的目的就是给自己增加信心。然后看了一些算法题。11月中旬开始面试，12月初入职。虽然没面试几家，但是还是得到了自己梦想中公司的offer，还是很开心的，虽然工资低了一些，但为了梦想嘛，值了！（虽然进去之后发现与预期天差地别）。

回北京的时候，自己还是处于一种异常的状态，感冒好长时间才好，总是头疼，也不知道是不是焦虑太大导致的，这种情况现在也偶尔会出现，怕不是烧坏了脑子了吧。过度的焦虑让我很煎熬，身体机能也跟着急速的下降。虽然说后来让自己走出了焦虑的阴霾，但是这期间的滋味真是不好受，身体也受到了一些损伤。

于是规律的健身又重新提上了日程，先从每次200个俯卧撑开始。而这次坚持多久我也不知道，我希望就这么一直下去吧，不存在坚持，把健身当成每天的必修课，把它当成吃饭喝水就好了。

加油吧！少年。年轻的你怎么知道未来在何方，焦虑又能解决什么问题，脚踏实地一步一个脚印，哪怕明天就不做程序员了，今天也不能放下这本Code秘籍。
