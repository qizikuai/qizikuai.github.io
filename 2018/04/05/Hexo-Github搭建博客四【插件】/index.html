Algolia 搜索

注册

前往 Algolia 注册页面，注册一个新账户。

注册完成后，创建一个新的 Index，这个 Index 将在后面使用。

安装

获取key

在 Algolia 服务站点上找到需要使用的一些配置的值，包括 ApplicationID、Search-Only API Key、 Admin API Key。注意，Admin API Key 需要保密保存。点击ALL API KEYS 找到新建INDEX对应的key， 编辑权限，在弹出框中找到ACL选择勾选Add records, Delete records, List indices, Delete index权限，点击update更新。

需要修改下面这三个参数：
官网上没有给apiKey: ‘apiKey’，我实际配置的时候发现不配置这个值得话，搜索按钮点击会没反应。

Config Key
applicationID Your Algolia Application ID
apiKey A Search-Only API key
indexName The name of the Algolia index to use

更新Index

将Search-Only API key 替换成Algolia中的对应的值
本地修改之后要使用hexo algolia 更新本地设置

主题集成

将 enable 改为 true 即可，根据需要你可以调整 labels 中的文本。

总结

  1. apiKey: ‘apiKey’一定要配置，这个参考地址
  2. 站点的_config.yml 中的url一定要配置成自己的站点地址，否则搜索之后点击跳转的不对，配置的跳转路径在Algolia网站上就能看到。
  3. Algolia 网站提供了很多统计信息，这个也是非常有用的。
